# Arma Mods Leaderboard

This project tracks and ranks **Arma Reforger** and **Arma 3** mods based on real server and player statistics using the BattleMetrics API.

## How it works?

1. A collector fetches all game servers from **BattleMetrics**.
2. Mod usage data is calculated — player counts, server counts, ranks, market share.
3. Results are served via API and displayed in a web dashboard with search, filtering, and trending analysis.

## Features

- **Mod Rankings** — hybrid ranking algorithm combining player and server counts
- **Trending Analysis** — which mods are rising or falling in popularity
- **History Timeline** — track mod performance over 24H, 7D, 30D, 90D, 1Y
- **Multi-game Support** — Arma Reforger and Arma 3
- **Server Details** — see which servers run which mods
